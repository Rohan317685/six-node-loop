# Analogical 

This is Analogical; a project I've made for chipless.hackclub.com, The main goal of this project was to learn analog and not use a microcontroller! 

It is a 6 node transistor only signal network where each block drives, perturbs and destabalises the next creating a really chaotic annalog behaviour which cannot be replicated with a MCU
